Install Python, Anaconda, Miniforge and JDK/JRE (only for pdf-xlsx).

Anaconda Prompt - https://www.anaconda.com/download/success

Miniforge - 
https://github.com/conda-forge/miniforge#miniforge3

Miniforge prompt:
> mamba create -n pyimagej pyimagej openjdk=11
> mamba activate pyimagej
> jupyter lab

Install those pip libraries, api and modules:

Anaconda prompt:
pdf-docx:
> pip install pdf2docx
txt-xlsx:
> pip install pandas
pdf-xlsx:
> pip install pandas
> pip install tabula-py
pdf-txt:
> pip install aspose-words
merge-pdf:
> pip install pypdf
